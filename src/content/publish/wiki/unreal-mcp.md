UnrealMCP는 Unreal Engine 작업을 AI 도구와 연결하려는 개인 R&D다.

공개 GitHub 기준 목표:

- MCP와 Claude Skill을 사용한 Unreal 제어
- 반복적인 에디터 작업 자동화
- 자연어 지시를 엔진 작업 단위로 변환
- 게임 개발 파이프라인에서 AI 도구의 역할 검증

이 주제는 [[unreal-client-programming]]과 연결된다. 클라이언트 개발 경험이 있어야 자동화 대상과 위험한 작업을 구분할 수 있다.
