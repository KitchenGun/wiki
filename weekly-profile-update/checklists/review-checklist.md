# Review Checklist

## Draft Review

- [ ] 고객사/조직명 노출 없음
- [ ] 서버/IP/토큰/계정 정보 없음
- [ ] 과장된 성과 표현 없음
- [ ] 공개 근거가 `source-evidence.md`에 있음
- [ ] 플랫폼별 문체가 맞음

## Asset Review

- [ ] 사용자가 Discord에 업로드함
- [ ] 사용자가 이미지 사용을 승인함
- [ ] 플랫폼별 비율이 적절함
